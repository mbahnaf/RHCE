# RHCE 8
RHCE with ansible 

## Pre-requisits    
-   ssh server configuration 
-   password and keybase authentication 
    -   sshkey-gen
    -   ssh-copy-id
    -   ssh private key ssh public key 
-   sudo configuration 
-   web server configuration 
-   enable / disable service 
-   start / stop service 
-   firewalld
